Easy Road

This is some of the process work for the Travel App, 'Easy Road'. This project is trying to address the need for a high level decision making solution to help people discover new travel locations by leveraging user reviews from other travelers and collating and representing them in an easy to read format. The process included user interviews, infinity maps, cardsorting, competitve audit, paper prototypes, lofi and hifi wireframes, prototyping and user testing.
